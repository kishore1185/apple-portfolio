#  Apple Vision–Style Portfolio

An immersive, motion-driven 3D portfolio built to showcase modern web development, UI design, and creative engineering.

🔗 Live Demo: https://kishoree.vercel.app/

---

## ✨ Overview

This portfolio is designed as a **next-generation interactive experience**, inspired by Apple Vision Pro aesthetics.

It blends:
- Smooth parallax motion
- Glassmorphism UI
- Real-time interactions
- Minimal yet powerful design

The goal is not just to show projects — but to **create an experience**.

---

## 🧠 Features

- 🌀 Smooth mouse-based parallax movement  
- 🎯 Unified motion system (background + UI sync)  
- 🧊 Glassmorphism cards with blur effects  
- 🎥 Immersive Apple-style UI layout  
- 📱 Fully responsive design  
- ⚡ Fast performance with optimized rendering  

---

## 🛠️ Tech Stack

- **React + TypeScript**
- **Vite**
- **Tailwind CSS**
- **Framer Motion**
- **Custom Parallax Engine**

## ⚙️ Installation

```bash
git clone https://github.com/kishore1185/apple-portfolio.git
cd apple-portfolio
npm install
npm run dev

🌐 Deployment
This project is deployed using Vercel

To deploy your own:
npm run build
```
